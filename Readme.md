# Campus Information Hub

## Project Description
A responsive single-page website providing essential campus information including services, locations, announcements, and layout demonstrations. Built to showcase modern CSS layout techniques, positioning, and responsive design.

## Features Demonstrated
- Fixed navigation bar with dropdown menus
- Responsive service cards using CSS Grid
- Image gallery with flexible layout
- Sticky notices that remain visible while scrolling
- Display behavior demo (visible, hidden, display: none)
- Responsive breakpoints for desktop, tablet, and mobile
- Semantic HTML structure
- Flexbox and Grid layouts
- Media queries for adaptability

## Layout & Responsiveness
- **Desktop**: Multi-column grids, full dropdowns
- **Tablet (≤768px)**: Hamburger menu, stacked cards
- **Mobile (≤480px)**: Compact spacing, single-column layouts

## Project Files
- `index.html`: Main HTML structure with all sections
- `style.css`: External styles with responsive design
- `README.md`: Project documentation
